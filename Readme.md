 Retail Sales Data Analysis Project
 Project Overview

This project focuses on end-to-end retail sales data analysis using Python.
The goal is to extract business insights from transactional data that can help decision-makers improve revenue, optimize product strategy, and control logistics costs.

The dataset was sourced from Kaggle and represents real-world retail pricing, sales volume, and freight costs.

 Objectives

Analyze monthly sales trends

Identify top-performing products

Understand category-wise revenue contribution

Explore the relationship between quantity sold and revenue

Evaluate the impact of freight costs on profit

 Tools & Technologies

Python

Pandas – data cleaning & manipulation

Matplotlib – data visualization

Jupyter Notebook 

 Data Cleaning & Preparation

Selected relevant columns for analysis

Renamed columns for clarity and consistency

Converted date columns to proper datetime format

Created new features such as Month and Year

Calculated Profit using Revenue and Freight Cost
